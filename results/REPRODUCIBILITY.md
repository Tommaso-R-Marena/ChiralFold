# ChiralFold — Results Reproducibility Guide

Every quantitative result in the README and the paper can be reproduced
from the scripts in `benchmarks/` and the data artefacts in this
directory. The instructions below walk through the five main results in
the order they are reported in the paper.

All numbers below are the values produced by the canonical run that
populated this directory and should match within rounding when
re-executed.

## 1. PDB-wide D-residue survey

**Script:** `benchmarks/d_residue_survey.py`

**Output:** `results/d_residue_verification.csv` (12,574 rows: 12,573 checkable D-amino acid residues + 1 incomplete-backbone row).

Drives the headline figures:
- 12,573 D-amino acid residues checked across 4,616 PDB files
- >91% coverage of all 18 standard D-amino acid CCD codes
- 1,209 files contained at least one D-amino acid

Reproduction:
```bash
python benchmarks/d_residue_survey.py
```
Expected wall time: 60–120 min the first run (PDB downloads).

## 2. Error classification

**Script:** `benchmarks/classify_d_errors.py`

**Output:** `results/error_classification.json`, `results/error_table_verified.csv`

Expected breakdown (from `error_classification.json`):
- 29 D-label/L-coordinate mismatches across 16 structures
- 0.23% error rate
- 6 genuine stereochemistry errors
- 18 CCD code misassignments (5 structures)
- 3 polymer residue mislabels
- 2 borderline cases

Reproduction:
```bash
python benchmarks/classify_d_errors.py
```

## 3. Independent verification (numpy only)

**Script:** `benchmarks/independent_d_residue_verification.py`

**Output:** `results/d_residue_verification_summary.json`

This is the primary independent validation: it imports **no ChiralFold
code**, only numpy and the standard library, so a reviewer can confirm
the survey result without trusting ChiralFold. Expected JSON contents:
- `total_d_residues`: 12,574
- `checkable_residues`: 12,573
- `d_correct`: 12,544
- `l_error`: 29
- `error_rate_pct`: 0.23

Reproduction:
```bash
python benchmarks/independent_d_residue_verification.py
```
Expected wall time: 10–30 s once the local PDB cache exists.

## 4. AF3 vs ChiralFold comparison (Childs et al. 2025)

**Script:** `benchmarks/childs2025_comparison.py`

**Output:** `results/childs2025_comparison.json`

Expected key values:
- 41 sequences (DP19, DP9, DP12 systems + variants + L controls)
- 478 chiral residues total
- 0 violations with ChiralFold
- 0.0 ChiralFold rate vs 0.51 AF3 rate (Childs et al. 2025)

Fisher's exact contingency table (canonical value from `results/summary.json`):

| | violations | non-violations |
|---|---:|---:|
| ChiralFold | 0 | 467 |
| AF3 (Childs) | 8,481 | 8,148 |

Fisher's exact p ≈ **6.66 × 10⁻¹⁴⁴**.

Reproduction:
```bash
python benchmarks/childs2025_comparison.py
```

## 5. MolProbity comparison

**Script:** `benchmarks/molprobity_comparison.py`

**Output:** `results/molprobity_comparison.json`, `results/molprobity_comparison.png`

Expected key values:
- 31 structures (X-ray + NMR + cryo-EM, 0.48–3.4 Å)
- Ramachandran outlier Spearman ρ ≈ 0.49 (p ≈ 0.006)
- ChiralFold mean outlier 0.60% vs wwPDB 0.64%
- 30/31 = 100% Cα correctness; 1 NMR structure (2JXR) flagged

Reproduction:
```bash
python benchmarks/molprobity_comparison.py
```

## End-to-end re-run

To regenerate the headline `results/summary.json` and figure set:
```bash
python benchmarks/run_full_benchmark.py
```

## Notes for reviewers

- The CSV and JSON artefacts in this directory are **frozen** outputs of
  the canonical run; they are checked into the repository so reviewers
  can verify counts without re-running.
- PDB coordinate files (`*.pdb`) included here (`1UBQ_D_mirror.pdb`,
  `1L2Y_D_mirror.pdb`, etc.) are intentional artefacts of the mirror
  pipeline benchmark — not deposited PDB entries.
- Do **not** edit the CSV/JSON files manually. They are regenerated by
  the scripts above.
