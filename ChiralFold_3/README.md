# ChiralFold_3/ — Preserved Version History

This directory is a **preserved version-history archive** of intermediate
ChiralFold v3.0–v3.2 development artefacts (LaTeX drafts and figures
used while iterating on the paper before v3.2.1).

It is **not** the production package. The current package lives in the
top-level `chiralfold/` directory, and the canonical paper source is at
`paper/chiralfold_paper.tex`.

## Contents

- `main.tex`, `main2.tex` — draft LaTeX manuscripts from earlier
  revisions, retained for historical reference.
- `*.jpg` — figure snapshots from the v3.0–v3.2 development cycle
  (`benchmark_results`, `comparison_table`, `folding_quality`,
  `mirror_binder_design`, `mirror_pipeline_benchmark`,
  `molprobity_comparison`, `survey_and_docking`, `Image`).

## Do not modify

Files in this directory exist only to document how the project evolved.
Reviewers comparing against the original v3.0–v3.2 figures can use them
as a baseline. Production code, paper, and metadata live elsewhere in
the repository.
